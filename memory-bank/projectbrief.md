# Project Brief: Tesera

## Overview
Tesera is an AI-powered operations assistant platform for SMEs, cooperatives, boutique e-commerce businesses, and hybrid physical-online sellers. It aims to automate customer communication, shipment management, stock tracking, workflow operations, and operational analytics using AI agents.

## Core Objectives
- Create a world-class AI operations platform.
- Solve SME operational problems with high automation.
- Deliver a modern, premium, calm, intelligent, and minimalistic UI/UX.

## Features
1. **AI Customer Communication:** WhatsApp/chat/email ready, AI-generated replies, order tracking automation.
2. **Inventory Management:** Low stock alerts, AI reorder suggestions, product tracking.
3. **Shipment Management:** Cargo tracking, delay detection, automated notifications.
4. **Workflow Automation:** Task generation, team assignments, automation rules.
5. **Analytics:** Sales trends, stock forecasting, operational insights.

## Scope & Constraints
- Only companies/businesses can register (no public consumer accounts).
- The dashboard must NEVER feel crowded.
- Minimal cognitive load.
- Mascot guides users during setup.

## Development Phases
- Phase 1: Project setup, Design system, Authentication, Landing page.
- Phase 2: Dashboard shell, Sidebar/topbar, Responsive layouts, User/company settings.
- Phase 3: Orders, Inventory, Shipments.
- Phase 4: AI assistant infrastructure, Agent workflows, Notifications.
- Phase 5: Analytics, Insights, Predictions.
- Phase 6: Production optimization, Docker, CI/CD preparation, Final polish.
